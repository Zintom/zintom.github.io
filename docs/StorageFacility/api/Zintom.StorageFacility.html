IRepresent<T>

An object may not be equal but may still 'represent' another object. For example, two boxes may differ in content, but on the outside still represent a 'box'.

Provides an interface to the Zintom.StorageFacility.Storage.StorageEditor, allowing you to put values into a Storage and also commit those values to file.
