Parameters
Type Name Description
System.Boolean outputOptimizedForReading

If true, the editor should try and optimize the output file for reading, using new-lines where appropriate.
Returns
Type Description
IStorageEditor
Remarks

The returned IStorageEditor persists for the life-time of this Storage object and can be called up through multiple calls to Edit(), the object returned is an identical reference to the underlying editor.

You must call Commit() to apply the changes to disk.
